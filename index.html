Rules

  • Setup: Draw an NxM grid (where N and M are the number of rows and columns, respectively).
  • Players: The game is typically played by two players.
  • Turns: Players take turns to add either an "S" or an "O" to an empty cell.
  • Scoring:Points are awarded for each occurrence of the word "SOS" formed by a player.
  • For example:if a player creates "SOS" horizontally in a row, they receive a point. The same goes for vertical and diagonal formations.
  • Winning Condition:The game continues until all cells in the grid are filled. The player with the most points at the end of the game wins.
  • Draw: If the entire grid is filled, and there is a tie in points, the game is declared a draw.
